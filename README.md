# :computer: Another Round(Only Code)

3D 원형맵 탄막류 로그라이트 게임(코드만)

# 📃 프로젝트 정보

### 1. 제작기간

> 2025/10/13 ~ 2025/11/10

### 2. 참여 인원

> |                    Name                    |  Position   |
> | :----------------------------------------: | :---------: |
> | [최정민](https://github.com/mista-che)   |  Back |

### 3. 역할 분담

> - 최정민 : 보스 패턴 구현, Player 이동 구현, Pad와 Keyboard를 사용한 입력 구현
> - 김동건 : 조명 및 안개효과 구현, 스킬 및 Reward UI 구현 
> - 박상영 : Enemy 3종 구현, 보스 애니메이션 추가, 사운드 구현
> - 전영배 : 회전하는 UI 구현, 로딩 화면 구현
